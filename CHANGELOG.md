## [1.0.0] - 2026-07-06
### Added
- Enterprise Sales Intelligence Platform launched.
- Core pages: Overview, Performance, Products, Customers, Intelligence, Board Reports.
- Machine Intelligence engine with explainable recommendations.
- Custom beautiful charts using Recharts with tailored Tooltips.
- 8pt Design System via CSS tokens and Tailwind.
- Subtle framer-motion micro-animations for enterprise delight.
